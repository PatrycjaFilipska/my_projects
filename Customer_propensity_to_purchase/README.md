# Customer Propensity to Purchase

## Business Problem

The company lacks a way to identify which customers are most likely to make a purchase.  
As a result, marketing campaigns are not targeted, leading to inefficient use of resources.

## Objective

The goal of this project is to build a predictive model that estimates the probability of a customer making a purchase, enabling more effective targeting and higher conversion rates.

---

## Dataset

The dataset contains customer-related features that can be used to predict purchasing behavior.

---

## Methodology

### 1. Data Preparation
- Handled missing values  
- Cleaned and standardized data  
- Prepared dataset for modeling  

### 2. Feature Engineering
- Selected relevant features  
- Transformed variables where necessary  

### 3. Modeling
- Built a classification model to predict purchase probability  

### 4. Model Evaluation
- Evaluated model performance using appropriate metrics  
- Assessed the model’s ability to distinguish between purchasing and non-purchasing customers  

---

## Key Insights

- The model can effectively identify customers with a higher likelihood of making a purchase  
- Not all customers have the same probability, highlighting the importance of segmentation  
- Targeting high-probability customers can significantly improve marketing efficiency  

---

## Business Recommendations

- Focus marketing campaigns on high-probability customers  
- Use model predictions to personalize offers  
- Reduce spend on low-probability segments  
- Continuously update the model with new data
