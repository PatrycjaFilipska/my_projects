# Data Analytics Portfolio

This repository contains a collection of data analysis projects focused on exploratory data analysis, data preparation, and practical insights extraction.

---

## Projects

### Customer Propensity to Purchase
Identification of customers who are most likely to make a purchase in the future.

###  Customer Segmentation (RFM)
Customer segmentation using Recency, Frequency, and Monetary analysis to identify key customer groups.

### Healthcare Dataset Analysis
Exploring a synthetic healthcare dataset containing patient records, including demographic data, clinical data, hospital information and financial data.

### Overstimulation Behavior Analysis
Conduct EDA and apply various statistical and machine learning methods to identify lifestyle behaviors, and habits that influence the likelihood of overstimulation in individuals.

### Shelter Dogs Analysis
Exploratory analysis of shelter dog characteristics, focusing on age, behavior, and physical traits.

### Spotify Analysis
Exploratory data analysis of music trends, including genre distribution, popularity patterns, and artist performance.
